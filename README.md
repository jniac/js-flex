# js-flex

resolving "kind-of-flex" model in javascript

## start developing
Watch changes on ./src (chokidar), build on changes (rollup), test build (mocha giveup, using custom lib).
```
npm start
```

## or simply test
```
npm test
```

## visual test

### local:

```
super-quick-static
```
